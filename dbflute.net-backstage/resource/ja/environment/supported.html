対応プログラミング言語

  • C#-3.0

補足

  • C#で自動生成されたクラスを、VB.NETのプログラムからライブラリとして利用することは可能です。
  • C#-3.0固有の機能は(積極的に)利用していないので、C#-2.0でも動作する可能性はあります(但し、将来は不明)。

対応データベース

  • MySQL *Exampleあり
  • PostgreSQL
  • Oracle *Exampleあり
  • DB2
  • SQLServer
  • SQLite *準サポート
  • MS Access *準サポート

対応DIコンテナ

  • Quill-1.3.x (Seasar.NET)

対応OS

DBFlute.NETの自動生成処理の実行環境として、以下の環境をサポートしています。

  • Mac OS X
  • Linux
  • Windows
