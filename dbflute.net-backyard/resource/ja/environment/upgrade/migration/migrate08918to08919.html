環境上の注意点

CBにて日付型のInScopeが生成される

CBにて日付型の InScope 条件が生成されるようになりました。日付で InScope を利用することはレアケースですので、コンパイルスピードを上げたい場合は(無駄なメソッドを生成しないようにしたい場合は)、includeQueryMap.dfprop にて、日付型の InScope、NotInScope の生成を抑制するようにして下さい。(抑制しても PK、FK カラムの InScope は強制的に生成されます) 

ex) 日付型のInScopeとNotInScopeを抑制する場合 @includeQueryMap.dfprop
    ; Date = map:{
        ; NotEqual = map:{}
        ; InScope = map:{}
        ; NotInScope = map:{}
    }
